# MCP 概览

## 设计目标

- 统一外部工具接入方式。
- 保证外部能力通过 Skills 抽象进入平台治理体系。

## 核心概念

- **MCP**：外部工具接入协议。
- **MCP Server**：提供工具能力的服务端。
- **MCP Adapter**：将 MCP 工具映射为 Skills 的适配层。

## 设计约束

- MCP 仅作为外部 Skill Provider。
- 所有 MCP 工具必须通过 Skill Runtime 调用。
- MCP 接入必须遵循权限、预算与审计要求。

## 非目标

- 不定义具体 MCP 网络协议实现细节。

## 未来演进方向

- 支持多协议兼容与跨版本协商。
